# Guía Aumentada - Museo Larco

Prototipo WebAR para enriquecer la visita al museo mediante realidad aumentada accesible desde el navegador.

## Instrucciones de Instalación
1. Clonar el repositorio.
2. Abrir la carpeta en VS Code.
3. **IMPORTANTE:** Usar la extensión "Live Server" para ejecutar el proyecto (click derecho en `index.html` -> "Open with Live Server"). Los navegadores bloquean la cámara si abres el archivo directamente haciendo doble click.

## Uso
1. Al abrir la web, conceder permisos de cámara.
2. Apuntar la cámara al marcador "Hiro".
3. Visualizar el modelo 3D, escuchar la narración y agregar a favoritos.

## Tecnologías
* **A-Frame:** Framework para escenas 3D/VR en web.
* **AR.js:** Motor de reconocimiento de marcadores eficiente para móviles.
* **HTML5/JS:** Lógica de interfaz y persistencia de datos (LocalStorage).
